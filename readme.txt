Hi. I'm a starter theme called Spotfin Starter Theme, based on _s. 

I'm used for projects over at http://spotf.in 

But you're welcome to use me and I encourage you to read my docs/faqs below.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

Spotfin Starter Theme removed the support of Jetpack's Infinite Scroll.

We support the use of Sizeable Sidebars and Sizeable Content Blocks.
